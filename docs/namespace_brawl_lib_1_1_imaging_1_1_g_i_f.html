Detailed Description

Class AnimatedGifEncoder - Encodes a GIF file consisting of one or more frames. 

Example:
   AnimatedGifEncoder e = new AnimatedGifEncoder();
   e.start(outputFileName);
   e.setDelay(1000);   // 1 frame per sec
   e.addFrame(image1);
   e.addFrame(image2);
   e.finish();

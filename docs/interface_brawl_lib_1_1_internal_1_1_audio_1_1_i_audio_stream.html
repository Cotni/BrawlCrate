int BrawlLib.Internal.Audio.IAudioStream.ReadSamples ( VoidPtr  destAddr,
int  numSamples

Reads numSamples audio samples into the address specified by destAddr. This method does not observe loop points and does not loop automatically.

Parameters
destAddrThe address at which to start writing samples.
numSamplesThe maximum number of samples to read.
Returns
The actual number of samples read (per channel).
